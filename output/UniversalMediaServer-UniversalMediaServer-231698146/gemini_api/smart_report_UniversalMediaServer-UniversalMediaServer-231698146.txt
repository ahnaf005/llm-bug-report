## Bug Report: `AudioProperties` Setters Do Not Throw `IllegalArgumentException` for Invalid Input

### 1. Title
`AudioProperties` setters fail to validate input, leading to `IllegalArgumentException` not being thrown as expected by tests.

### 2. Steps to Reproduce
1.  Clone the UniversalMediaServer repository at commit `0579589d0ccb96c37872eab4f6ba3238bcbe6fe6`.
2.  Ensure Java 7 and Maven 3.2.5 are installed.
3.  Run the Maven build command: `mvn install -Dmaven.javadoc.skip=true -B -V`
4.  Observe the build failure during the `maven-surefire-plugin:2.19.1:test` phase.

### 3. Triggering Input
The `AudioPropertiesTest` attempts to set audio properties (bitrate, number of channels, bits per sample, sample frequency) with invalid input values (specifically, values less than 1). The tests expect an `IllegalArgumentException` to be thrown in these scenarios.

### 4. Expected Behavior
When methods like `setBitRate`, `setNumberOfChannels`, `setBitsperSample`, and `setSampleFrequency` in `AudioProperties.java` are called with values less than 1, they should throw an `IllegalArgumentException` to indicate invalid input.

### 5. Observed Behavior
The build fails because the `AudioPropertiesTest` expects `IllegalArgumentException` to be thrown when invalid values (less than 1) are passed to the setter methods. However, the original implementation of these methods does not perform this validation, thus not throwing the expected exception. This causes the tests to fail with an `AssertionError` indicating that the expected exception was not thrown.

### 8. Patches / Suggested Fixes
The provided code diff contains the necessary patches to fix this bug. The fix involves adding `if` conditions to the setter methods in `AudioProperties.java` to check if the input value is less than 1. If it is, an `IllegalArgumentException` is thrown with a descriptive message.

Specifically, the following lines were added:
*   In `setBitRate(int bitRate)`:
    ```java
    if (bitRate < 1) {
        throw new IllegalArgumentException("Bitrate can't be less than 1.");
    }
    ```
*   In `setNumberOfChannels(int numberOfChannels)`:
    ```java
    if (numberOfChannels < 1) {
        throw new IllegalArgumentException("Channel number can't be less than 1.");
    }
    ```
*   In `setBitsperSample(int bitsperSample)`:
    ```java
    if (bitsperSample < 1) {
        throw new IllegalArgumentException("Bits per sample can't be less than 1.");
    }
    ```
*   In `setSampleFrequency(int sampleFrequency)`:
    ```java
    if (sampleFrequency < 1) {
        throw new IllegalArgumentException("Sample frequency can't be less than 1 Hz.");
    }
    ```
